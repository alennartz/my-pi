description: Survey repo state, summarize changes, and plan cleanup
---

Check the current git repository state and help me get it clean.

1. **Survey** — run whatever git commands are needed to build a full picture: staged changes, unstaged modifications, untracked files, stashes, and local commits not yet pushed to origin.

2. **Summarize** — group related changes and describe what they do at a high level — the intent, not the file list. Keep it brief.

3. **Propose** — for each group, one line: what to do (commit with message, gitignore, delete, stash, push). Present the whole plan as a short numbered list and ask for a yes/no before executing.
